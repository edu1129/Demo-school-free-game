इमेज अपलोड और URL जनरेशन

डैशबोर्ड पर वापस जाएं
मौजूदा इमेज URL प्रक्रिया

जब कोई उपयोगकर्ता फोटो अपलोड करता है, तो वह फोटो सीधे Google स्प्रेडशीट में सेव नहीं होती है। इसके बजाय, फोटो को GitHub पर अपलोड किया जाता है, और केवल उस फोटो का लिंक (URL) स्प्रेडशीट में सेव होता है।

संबंधित फ़ाइल और फ़ंक्शन

यह पूरी प्रक्रिया Code.gs फ़ाइल के अंदर uploadImageToGitHub नामक फ़ंक्शन द्वारा नियंत्रित की जाती है।

प्रक्रिया के चरण (Step-by-Step Process)
  1. GitHub से कनेक्शन: स्क्रिप्ट आपके GitHub खाते से जुड़ती है।
  2. यूनिक फ़ाइल पाथ बनाना: हर इमेज के लिए एक यूनिक पाथ बनाया जाता है (जैसे: /school_images/School_Name/timestamp_filename.jpg)।
  3. GitHub API पर अपलोड: स्क्रिप्ट इमेज को GitHub पर भेजती है।
  4. URL प्राप्त करना: सफल अपलोड पर, GitHub एक डायरेक्ट डाउनलोड URL (download_url) देता है।
  5. स्प्रेडशीट में सेव करना: यही URL स्प्रेडशीट के PhotoURL कॉलम में सेव हो जाता है।
कस्टम डोमेन URL कैसे सेट करें? (जैसे: https://image.digitalerp.shop/...)

आपका वांछित URL (https://image.digitalerp.shop/...) प्राप्त करने के लिए, आपको केवल कोड बदलने से ज़्यादा कुछ करना होगा। यह एक दो-चरणीय प्रक्रिया है:

चरण 1: अपना डोमेन और होस्टिंग सेट करें

सबसे पहले, आपको यह सुनिश्चित करना होगा कि आपका डोमेन image.digitalerp.shop उस स्थान पर इंगित (point) कर रहा है जहाँ आपकी इमेज संग्रहीत हैं।

  • विकल्प A (GitHub Pages): आप अपनी GitHub रिपॉजिटरी के लिए GitHub Pages को सक्रिय कर सकते हैं और उस पर अपना कस्टम डोमेन (image.digitalerp.shop) सेट कर सकते हैं।
  • विकल्प B (अन्य होस्टिंग): आप एक अलग इमेज होस्टिंग सेवा (जैसे AWS S3, Cloudinary, या कोई अन्य वेब होस्टिंग) का उपयोग कर सकते हैं और अपने डोमेन को वहां इंगित कर सकते हैं।
यह एक एडवांस सेटअप है जिसके लिए डोमेन और DNS सेटिंग्स के ज्ञान की आवश्यकता होती है।
चरण 2: Code.gs में बदलाव करें

एक बार जब आपका डोमेन सही ढंग से सेट हो जाए, तो आपको Code.gs फ़ाइल में uploadImageToGitHub फ़ंक्शन को संशोधित करना होगा। आपको GitHub से मिले URL को अनदेखा करना होगा और अपना खुद का URL बनाना होगा।

बदलाव का क्षेत्र: uploadImageToGitHub फ़ंक्शन के अंत में।

संक्षेप में, आपको पहले अपने कस्टम डोमेन को अपनी इमेज स्टोरेज से जोड़ना होगा और फिर Code.gs में कोड को संशोधित करना होगा ताकि वह GitHub द्वारा दिए गए URL के बजाय आपके कस्टम डोमेन का उपयोग करके URL बनाए।
